Apttus Digital Commerce Summer '20

Steps to upgrade Winter-2019 p-com template to consume Spring-20 SDK

1. Update Apttus Libraries

The following packages should be updated to the latest version:

  • @apttus/core ^1903.x.x --> ^2001.x.x
  • @apttus/ecommerce ^1903.x.x --> ^2001.x.x
  • @apttus/elements ^1903.x.x --> ~2001.x.x

Once done, your package.json should have the following packages under dependencies:

    ...
    "@apttus/core": "^2001.x.x",
    "@apttus/ecommerce": "^2001.x.x",
    "@apttus/elements": "^2001.x.x",
    ...

2. Replace ConstraintIconModule with ConstraintRuleModule

Open the "app.module.ts" file and remove the ConstraintIconModule from the imports and add ConstraintRuleModule.

3. Cart List Component Fix

The CartService no more has the getMyCarts method and instead we should use the generic query method. Replace the getMyCarts method as below,

this.cartService.query({
  page: {
    pageLimit: this.limit,
    pageNumber: this.currentPage
  }
})

4. Primary Contact Output Fix

The PrimaryContactId has been changed to _PrimaryContactId in the Quote model due to schema. To make the ouput field component in quote details page to display the primary contact correctly you need to change field value to "_PrimaryContactId" as shown below.

<apt-output-field [record]="quote" field="_PrimaryContactId" layout="inline"
            [editable]="quote?.Approval_Stage === 'Draft' || quote?.Approval_Stage === 'Presented'" 
            [showQuickView]="true" [lookupOptions]="lookupOptions"></apt-output-field>

5. Cart Model Changes

The Quote property in the Cart model is changed to Proposald. Please update the files under "carts" folder like "cart-table.component.ts", "cart-detail.component.ts" etc. to fix the issue.

6. Route Fixes

The route to assets is changed from "/asset-list" to "/assets". Update the files like "app-routing.module.ts", "header.component.html" and wherever the old route is used to solve the routing problem.

The route to active cart is wrongly specified as "/cart" in "quote-detail.component.ts" which has to be replaced with "/carts/active".

this.router.navigate(['/carts', 'active']);
